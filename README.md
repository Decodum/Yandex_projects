# 🧠 Проекты по Data Science (Яндекс.Практикум)

Этот репозиторий содержит учебные проекты, выполненные в рамках курса **"Специалист по Data Science"** от **Яндекс.Практикума**. В ходе обучения я работал над решением бизнес-задач с использованием методов машинного обучения, анализа данных и визуализации.

## 📂 Содержание

| Файл | Описание проекта |
| :--- | :--- |
| `apartments_sales.ipynb` | **Исследование рынка недвижимости:** Анализ базы данных сервиса Яндекс.Недвижимость. Цель — выявить ключевые факторы, влияющие на стоимость квартир в Санкт-Петербурге и Ленинградской области. |
| `Gold_recovery.ipynb` | **Модель восстановления золота:** Разработка модели машинного обучения для промышленной компании. Задача — предсказать коэффициент восстановления золота из руды, что позволяет оптимизировать производство и избежать убытков. |
| `Taxi_prediction.ipynb` | **Прогнозирование заказов такси:** Построение модели, которая предсказывает количество заказов такси на следующий час (временные ряды). Решение позволяет компании «Чётенькое такси» эффективно управлять ресурсами в пиковые часы. |
| `Telecom_project.ipynb` | **Прогнозирование оттока клиентов (Churn Prediction):** Анализ данных телеком-оператора и построение модели, которая выявляет клиентов, планирующих расторгнуть договор. Это помогает удерживать пользователей с помощью специальных предложений. |

## 🛠️ Используемые инструменты

Проекты выполнены на языке **Python** с использованием популярных библиотек Data Science:

*   **Обработка данных:** `Pandas`, `NumPy`
*   **Визуализация:** `Matplotlib`, `Seaborn`
*   **Машинное обучение:** `Scikit-learn` (линейная регрессия, случайный лес, градиентный бустинг, логистическая регрессия)
*   **Анализ временных рядов:** `Statsmodels`
*   **Среда разработки:** `Jupyter Notebook`
